##简介  
apino是一个免费开源的、快速、简单的轻量级的PHP渐进式框架。  
宗旨以最少的代码实现更高效的功能

##功能  
1.apino表达式无需编写后端代码，URL以Restful风格为核心  
2.预设多API，可自定义参数进行更改  
3.当apino表达式无法满足需求时，支持用户自定义  
4.内置URL权限控制
